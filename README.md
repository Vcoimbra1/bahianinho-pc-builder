# Bahianinho PC Parts Configurator

Um simulador de montagem de hardware focado em compatibilidade, desenvolvido com a energia e o estilo do Nordeste! 🚀

## Funcionalidades
- **Filtros Dinâmicos:** Selecione uma CPU e veja apenas as placas-mãe compatíveis.
- **Validação de Memória:** O sistema avisa se você tentar colocar DDR4 em uma placa DDR5.
- **Cálculo de Energia:** Somatório automático de TDP com sugestão de fonte ideal (margem de segurança de 30%).
- **Gabinete Inteligente:** Verificação de formato (ATX vs Micro-ATX).

##  Tecnologias Utilizadas
- **React 18** 
- **Tailwind CSS v4** 
- **JavaScript ES6+**
